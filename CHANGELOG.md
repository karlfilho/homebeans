# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-28

### Added
- Double-entry personal accounting engine in Python.
- YAML-based local ledger storage.
- Interactive CLI for adding transactions.
- MCP server integration for AI assistants.
- Financial reports including balances, statements, cashflow, and summaries.
- Demo mode with fictional data.
- Guided tutorial in demo mode.
- Account suggestions based on prior usage.
- MIT license.

### Tested
- 91 automated tests covering models, double-entry validation, demo mode, MCP tools, and reports.
